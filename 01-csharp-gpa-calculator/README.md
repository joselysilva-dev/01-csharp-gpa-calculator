# 🎓 GPA Calculator - C#

Simple GPA Calculator developed with C# and .NET 8.

## 📚 About

This project was created as part of my Software Engineering learning journey.

The application calculates a student's GPA based on grades entered through the console.

## 🚀 Technologies

* C#
* .NET 8
* Git
* GitHub

## 💻 Features

* Student name input
* Grade input
* GPA calculation
* Formatted console output

## ▶️ Run

```bash
dotnet run
```
